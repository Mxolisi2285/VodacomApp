# Vodacom Console App

A **C# console application** that simulates buying **data**, **voice minutes**, or a **data + voice bundle**, similar to Vodacom services.  

This project is built using **.NET 10** and demonstrates basic C# programming concepts including user input, conditional statements, loops, methods, input validation, and basic console UI design.

---

## Features

- Buy **Data** (GB)  
- Buy **Voice Minutes**  
- Buy **Data + Voice Bundle** (with discounted rates)  
- Exit the application safely  
- Input validation to prevent crashes
